Projects

  • Avalanche Prediction

    The "Avalanche Prediction" initiative employs a DecisionTreeClassifier ML algorithm to anticipate avalanches, considering variables like elevation, wind speed, humidity, and temperature. Seamlessly integrated with Flask, it produces predictions displayed on a user-friendly web interface. This pioneering approach strengthens safety protocols in avalanche-prone areas, offering crucial insights for prompt intervention. Skills: Flask · Python (Programming Language) · Machine Learning · Web Development · Bootstrap (Framework)

  • CV Extractor

    CV Extractor is a streamlined solution for effortlessly extracting key details from resumes and CVs. Using Streamlit for a user-friendly interface, users can upload documents with ease. Behind the scenes, sophisticated text processing techniques efficiently identify and extract email addresses, phone numbers, and other text. The output, typically in CSV format, enables seamless integration with various systems. This project streamlines recruitment workflows, saving time and improving accuracy in candidate evaluation. Skills: Python (Programming Language) · streamlit
